Welcome to GitHub Pages.

Instruction on how to work with Bitbucket or Github:

Remember to use Chocolatey to install Git client. This way, you won't have to deal with program files scattered all over your hard drive. Chocolatey manages them for you.

Git package on Chocolatey; http://chocolatey.org/packages/git.install

The first time you use a repo, you need to clone it: 


git clone https://<your account name>@bitbucket.org/vpulse/ecommerce.git

Next, go change some files.

Then:

to stage the code for commit: (from the root of the repo directory, this will stage everything that's changed"


git add .

then commit the changes to your local repo:


git commit -m "JIRA_TICKET:  check in comments here"

never check code in without a comment, and JIRA number is required for our team.

to "push" your committed repo to the remote repo:


git push

It will prompt you for your sign in credential. that's how you know your code is being synced with a remote repo.

To get changes from other people:


git fetch


git status


git pull

Remember, 


git status

is your friend. You can learn a lot by reading the status message.

Please learn git by using these simple command lines as above, until you feel comfortable with git (takes 2-3 days) After that, use any graphic tools you like. there are many good ones. with an understanding of the commands, it will be much more productive to use a tool because you will understand what the tool is doing under the cover.
